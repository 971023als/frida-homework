### 2️⃣ **자동화 공격 프로젝트**
1. **Blind SQL Injection**
   - **개발 환경**:  
     - **언어**: Java (프레임워크 활용 가능)  
     - **DBMS**: Oracle
   - **과제 목표**:
     - Blind SQL Injection 취약점 발생 환경 구현
     - DBMS 데이터 추출을 위한 **Python 기반 자동화 스크립트** 개발
     - 공격 속도 최적화를 위한 알고리즘 개선
     - **PoC (Proof of Concept)** 과정 상세 기술
   - **특별 과제**:
     - SQLi 발생 구간이 **SELECT 절 이외의 구간(UPDATE/DELETE)**일 경우,  
       - 다른 데이터에 영향을 주지 않으면서 공격을 수행하는 방안 도출 및 기술
